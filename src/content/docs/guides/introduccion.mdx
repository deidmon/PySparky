---

PySpark es la API de Python para Apache Spark, un potente motor de cálculo distribuido de código abierto. Permite a los desarrolladores y científicos de datos aprovechar la potencia del procesamiento en paralelo y distribuido de Spark utilizando un lenguaje tan popular y versátil como Python.

Esta guía está diseñada para ser tu compañera de viaje en el aprendizaje de PySpark, desde los conceptos más básicos hasta las técnicas más avanzadas.

### **¿Qué aprenderás en esta guía?**

* **Fundamentos sólidos:** Comprenderás la arquitectura de Spark y los conceptos clave como RDDs, DataFrames y Spark SQL.
* **Manipulación de datos:** Aprenderás a realizar transformaciones y acciones para limpiar, filtrar, agregar y analizar grandes conjuntos de datos de manera eficiente.
* **Funcionalidades avanzadas:** Explorarás temas como las funciones de ventana, el manejo de datos particionados y la creación de funciones definidas por el usuario (UDFs).
* **Buenas prácticas:** Descubrirás cómo optimizar tu código, manejar errores y escribir tests para asegurar la calidad de tus aplicaciones Spark.

¡Empecemos! Navega por las secciones de la izquierda para comenzar tu viaje en el mundo de PySpark.
